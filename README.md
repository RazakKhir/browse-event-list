# Summary

Creating a UI to browse event list using React JS.

Tools/Library used for this code:
  - Intellij IDEA
  - Node v14
  - NPM v6
  - ReactJS
  - Antd / SCSS
  
#### To run/edit this on your local;

  1. Clone this project
  2. Run:
  ```
    npm install
  ```
  3. Then run:
  ```
    npm start
  ```
  4. Happy coding!

Deployed application -> https://razakkhir.github.io/browse-event-list/
